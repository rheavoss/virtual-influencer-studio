# Omnichannel Distribution Matrix
**Strategy:** Create Once, Publish Everywhere (COPE). The goal is to maximize leverage by generating a core piece of media (4:5 image or 9:16 video) and syndicating it across every algorithm with zero extra production cost.

## 1. Video Distribution (The 9:16 Vertical Domination)
A single vertical (9:16) video can be natively pushed to 6 different major algorithms simultaneously.

| Platform | Format Name | Orientation | Resolution | Ideal Length | Link Capability |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Instagram** | Reels | 9:16 (Vertical) | 1080x1920 | 5-15s | Link in Bio / Stories |
| **Facebook** | Reels | 9:16 (Vertical) | 1080x1920 | 5-15s | High (Comments, Profile) |
| **TikTok** | Feed Video | 9:16 (Vertical) | 1080x1920 | 7-15s | Link in Bio (once unlocked) |
| **YouTube** | Shorts | 9:16 (Vertical) | 1080x1920 | 10-60s | Pinned Comment Link |
| **Snapchat** | Spotlight / Stories | 9:16 (Vertical) | 1080x1920 | 5-10s | Swipe Up Link |
| **Pinterest** | Idea Pins | 9:16 (Vertical) | 1080x1920 | 15s | Direct Outbound Link |

---

## 2. Image Distribution (The 4:5 Aesthetic Domination)
A single high-quality portrait (4:5) image commands maximum screen real estate on mobile feeds.

| Platform | Format Name | Orientation | Resolution | Behavior / Value |
| :--- | :--- | :--- | :--- | :--- |
| **Instagram** | Feed Post | 4:5 (Portrait) | 1080x1350 | Core aesthetic building, saves, tags |
| **Facebook** | Profile / Group Post | 4:5 or 1:1 | 1080x1350 | Text-heavy captions, sharing to groups |
| **X (Twitter)** | Media Feed | Any (crops to 16:9 in feed preview, opens to full) | 1080x1350 | Uncensored reach, direct funnel to Fanvue |
| **YouTube** | Community Tab | 1:1 (Square) or 4:5 | 1080x1080 | Highly pushed to non-subscribers to drive channel growth |
| **Telegram** | Channel Post | Any | Full Res | VIP un-algorithmic direct connection |
| **Reddit** | Subreddit Post | Any | Full Res | Massive top-of-funnel viral traffic |

---

## 3. The Paywall Distribution (Fanvue / Subscription)
This is where the traffic is harvested. The paywall layer receives the *extended* or *uncensored* version of the public content.

| Platform | Format Name | Orientation | Content Type | Monetization Logic |
| :--- | :--- | :--- | :--- | :--- |
| **Fanvue / Patreon** | PPV (Pay Per View) Message | Any | Image/Video/Audio | The "Tease & Reward" loop. |
| **Fanvue / Patreon** | Feed Post | Any | Image/Video | Exclusive subscriber retention content. |
| **Facebook** | Subscriber Only Hub | 9:16 or 4:5 | Image/Video | Native monetization without clicking off-platform. |

## Maximum Efficiency Rule:
**Never render in 16:9 (Landscape) or 1:1 (Square) natively.** 
Always render in **4:5 for Images** and **9:16 for Video**. 
These formats command 100% of the mobile screen height, maximizing attention span and engagement across meta, byte-dance, and google mobile ecosystems.
