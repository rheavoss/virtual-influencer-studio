## Day-by-Day Protocol

### Day 1 — Account Creation + Setup
- Create account via physical device + Calilio US number
- Username: `@sonia.mako`
- Bio: Blank for now (no links Day 1)
- Profile photo: First Sonia image (SFW, non-AI-looking)
- Action: Follow 30–50 accounts (fitness, beauty, lifestyle — NOT GFE)
- **NO POSTS. NO LINKS.**

### Day 2 — Warming
- Like 30–50 posts in explore feed (fitness, fashion, lifestyle)
- Watch 10+ Reels fully (signals active engagement to algorithm)
- Leave 5 genuine comments on non-GFE accounts
- **NO POSTS YET.**

### Day 3 — First Post
- Post 1 SFW Sonia photo (casual, non-AI-looking style)
- Caption: personal, relatable tone ("finally took a break from the gym 😅")
- NO external links in caption or bio yet
- Watch comments/likes for first 2 hours — reply to everything

### Day 4 — Build Rhythm
- Post 1 more SFW photo
- Like/comment on 20+ posts
- Follow 20 more accounts
- Add link.boo URL to bio (soft, no announcement)

### Day 5 — Engagement Push
- Post 2 photos (different times: 10am + 7pm)
- First soft CTA in one caption: "more on my page 🔗 link in bio"
- Engage with anyone who followed you (like their content)
- Report: track follower count baseline

### Day 6 — Reel Introduction
- Post first short Reel (15 seconds, no CTA, pure aesthetic)
- Continue photo posts (1 photo + 1 reel = 2 posts today)
- Engage 30+ accounts

### Day 7 — Full Speed
- Post 3 times (2 photos + 1 reel)
- One post with clear CTA: "exclusive content on Fanvue — link in bio 💋"
- Pin best-performing post from the week
- Check Instagram Insights — note reach, impressions, profile visits

---

## What NOT to Do (Ban Prevention)

| ❌ Avoid | Reason |
|---|---|
| Using emulators or virtual devices | IG device fingerprinting |
| Multiple accounts on same device | Account linking |
| Following 200+ people/day | Spam flag |
| Posting links Day 1–3 | Trust score killer |
| Buying followers | Instant shadow ban |
| Using external scheduling tools Day 1–7 | Triggers spam detection |
| Posting very similar images back-to-back | Duplicate content flag |

---

## IG Algorithm Signals to Build

| Signal | How to Build |
|---|---|
| Session time | Watch Reels fully before posting |
| Comment quality | Give thoughtful comments, not emoji-only |
| Save rate | Create content worth saving |
| Share rate | Create content worth sending to friends |
| Profile visits | Strong bio + consistent aesthetic |

## Post-Warm-Up Cadence (Week 2+)

- 3–5 Reels/week minimum
- 1–2 static photos/week
- Stories: 2–3 daily
- Never go 48+ hours without posting (algorithm penalizes dormancy)
