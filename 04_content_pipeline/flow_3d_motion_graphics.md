- Story-style animated frames

## Process

1. **Generate base video** — via Higgsfield or Hyperframes
2. **Open Flow** — upload base video
3. **Apply master prompt** — ChatGPT generates the right Flow prompt for the content type
4. **Render** — Flow adds 3D elements on top
5. **Export** — Final video ready for upload

---

## Master Prompt Template (ChatGPT)

```
I have a [content type: IG reel / teaser / story] featuring [brief description].
I want to add professional 3D motion graphic overlays using Flow.
Generate a Flow-compatible prompt that adds: [specific effects needed].
Style: premium, sensual, minimalist. Colors: [based on content palette].
Output: exact Flow prompt I can paste directly.
```
