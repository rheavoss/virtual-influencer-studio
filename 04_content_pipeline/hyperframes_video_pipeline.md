## What Is It

Write simple HTML compositions using data attributes → agent renders them into studio-quality short-form videos.

Deterministic = same HTML input always produces the same output. No randomness, no drift.

**Relevance Score: 9.8/10** — highest consistency for short-form content factory.

---

## Why Deterministic Matters for Jasmine

Normal AI video: different results every generation → face drift, body inconsistency, wasted credits.

Hyperframes: HTML describes exactly what happens → consistent output every time → fast reel factory.

---

## How It Works

```html
<!-- Example Hyperframes composition -->
<scene duration="3s" background="beach sunset">
  <character ref="jasmine_reference.jpg" pose="standing" expression="soft smile" />
  <text position="bottom" style="white bold">link in bio 💋</text>
  <audio src="trending_audio.mp3" />
</scene>
```

Agent reads this HTML → renders video with Jasmine in exact pose/expression/setting.
